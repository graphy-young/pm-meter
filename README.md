# pm-meter
source for research projects

# Requirements
### Hardware
* Raspberry Pi 3 or Newer (3 model recommended for less voltage use)
* Honeywell HPMA115S0-XXX
* database server to record measured data

### Software
* Raspbian OS or Debian Linux
* Python 3
* mysql

### Library
* honeywell_hpma115s0
* os
* pymysql
